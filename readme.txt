Extend your WordPress with social oriented voice comment system.
== Description ==
This plugin integrates build-in text comments with modern voice comment system. All files are hosted on uemotion.com, so you don't need big hard drive. You can adjust appearance to your needs.
== Installation ==
1. Download the plugin
2. Upload the wpue folder to your site's wp-content/plugins directory
3. Activate the plugin in the WordPress Admin Panel
4. Configure plugin in the WordPress Admin Panel
